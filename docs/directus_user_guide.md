## 3. Редактирование контента сайта

### Коллекция «Контент сайта» (site_content)

Эта коллекция содержит все текстовые элементы сайта.

**Как редактировать:**
1. Нажмите на **«Контент сайта»** в боковой панели
2. Найдите нужный элемент по колонке **Description** (описание)
3. Кликните на строку для редактирования

![Форма редактирования контента](/Users/apple/.gemini/antigravity/brain/c38d34cc-7e33-437e-b0cb-d9b6fb3f4cd1/directus_site_content_edit_form_1769077034066.png)

### Поля для редактирования:

| Поле | Описание |
|------|----------|
| **Key** | Технический идентификатор (НЕ РЕДАКТИРОВАТЬ!) |
| **Value** | Текст, который отображается на сайте |
| **Description** | Подсказка, где используется этот текст |
| **Type** | Тип контента |

> [!CAUTION]
> Никогда не меняйте поле **Key** — это может сломать отображение на сайте!

### Примеры ключей контента:

| Key | Где отображается |
|-----|------------------|
| `ui_nav_company` | Пункт меню «О компании» |
| `buyer_title` | Заголовок Hero-блока для покупателей |
| `buyer_subtitle` | Подзаголовок Hero-блока для покупателей |
| `company_name` | Название компании |
| `company_title` | Заголовок страницы «О компании» |

## 8. Часто задаваемые вопросы

### Как изменить заголовок на главной странице?
1. Откройте **«Контент сайта»**
2. Найдите записи с ключами `buyer_title`, `seller_title`, `engineer_title`
3. Измените поле **Value**

### Как добавить новое решение?
1. Откройте **«Решения»**
2. Нажмите кнопку **+** в правом верхнем углу
3. Заполните все поля
4. Сохраните

### Как изменить контактную информацию?
1. Откройте **«Офисы»**
2. Выберите офис
3. Измените нужные поля

### Изменения не отображаются на сайте?
- Подождите 1-2 минуты (кэширование)
- Обновите страницу (Ctrl+F5)
- Очистите кэш браузера
