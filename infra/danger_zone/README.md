# ОПАСНАЯ ЗОНА (DANGER ZONE)

В этой директории находятся скрипты, которые могут привести к **временной остановке всех сервисов** или **потере несохраненного состояния контейнеров**.

### Скрипты:
1. `nuclear_reset.exp` — Полная перезагрузка стека с пересозданием сетевых интерфейсов. Использовать ТОЛЬКО если возникли критические ошибки в работе Docker Network.
2. `hard_reset.exp` — Жесткий рестарт всех контейнеров.

### Инструкции:
- Не запускайте эти скрипты, если не уверены на 100% в необходимости.
- Всегда имейте свежий бэкап базы данных перед запуском.
